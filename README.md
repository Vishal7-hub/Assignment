Brief Description

This project is a modern, interactive hero section built using React and GSAP, designed to deliver a smooth, scroll-based animation experience.

On initial load, a letter-spaced headline (“WELCOME ITZFIZZ”) appears with a staggered animation, followed by animated statistics. As the user scrolls, a car image moves smoothly from left to right across the screen, synchronized with scroll progress using GSAP ScrollTrigger.

The project focuses on performance, smooth motion, and premium UI/UX by using transform-based animations, parallax background effects, and responsive design techniques.

Key Features
Smooth scroll-driven animation using GSAP
Staggered text reveal on page load
Animated statistics section
Car movement synced with scroll progress
Parallax background effect for depth
Fully responsive and optimized performance

Tech Stack
React.js (Vite)
GSAP (ScrollTrigger)
HTML5
CSS3
